# Design Variants Index

Tato složka obsahuje varianty designu pro různé části Bydlo prototypu. Každá oblast má vlastní podsložku s README indexem.

## Oblasti

- **[landing-page/](variants/landing-page/)** — Landing page CTA, intro text, vstupní formulář
- **[conversation/](variants/conversation/)** — Conversation flow, otázky, tón
- **[results/](variants/results/)** — Results page layout, filtering, sorting
- **[designer-profile/](variants/designer-profile/)** — Designer detail page layout

## Stav variant

- ✅ **Implementováno** — živá verze v kódu
- 🧪 **V testování** — testuje se s uživateli
- 📝 **Návrh** — navrženo, čeká na rozhodnutí
- ❌ **Zamítnuto** — otestováno a zamítnuto
